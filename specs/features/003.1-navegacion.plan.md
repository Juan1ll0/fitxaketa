# Plan de Implementación: 003.1 - Navegación Bottom Nav y Layout Compartido

## Resumen

Implementar sistema de navegación por tabs inferior (BottomNav) con 3 secciones principales, actualizar el layout raíz para incluirlo, y extender el store global `app-state.ts` para centralizar el estado de la aplicación (jornadas cargadas, periodo seleccionado, etc.). Esta es la base estructural para las demás sub-specs del dashboard.

## Riesgos Técnicos

### 1. Padding-bottom insuficiente

**Riesgo:** El contenido puede quedar oculto bajo el bottom nav si el padding es muy pequeño.
**Mitigación:** Usar `pb-20` (5rem = 80px) que es mayor que `h-16` (4rem = 64px) del nav, dejando margen de seguridad.

### 2. Detección incorrecta de tab activo

**Riesgo:** La función `isActive()` puede no detectar correctamente rutas anidadas.
**Mitigación:** Usar `startsWith(href)` para rutas distintas de `/`, y comparación exacta para `/`.

### 3. Iconos SVG inconsistentes

**Riesgo:** Iconos con diferentes tamaños o estilos visuales.
**Mitigación:** Definir todos los SVGs con `width="24" height="24"` y `stroke-width="2"` para consistencia.

### 4. Client-side routing no funciona

**Riesgo:** Los enlaces `<a>` pueden causar recarga completa en lugar de navegación SPA.
**Mitigación:** SvelteKit maneja automáticamente la navegación client-side para enlaces `<a>` internos. Verificar que no haya atributos `rel="external"`.

### 5. Accesibilidad incompleta

**Riesgo:** Navegación con teclado no funciona correctamente.
**Mitigación:** Usar elementos `<a>` nativos (son focusables por defecto) y verificar con pruebas manuales de Tab/Enter.

## Notas de Implementación

### Estructura de BottomNav.svelte

```svelte
<script lang="ts">
	import { page } from '$app/stores';

	const tabs = [
		{ href: '/', label: 'Fichar', icon: 'clock' },
		{ href: '/historial', label: 'Historial', icon: 'list' },
		{ href: '/estadisticas', label: 'Estadísticas', icon: 'chart' }
	];

	function isActive(href: string): boolean {
		if (href === '/') return $page.url.pathname === '/';
		return $page.url.pathname.startsWith(href);
	}
</script>

<nav
	class="fixed bottom-0 left-0 right-0 bg-surface border-t border-border"
	aria-label="Navegación principal"
>
	<ul class="flex justify-around items-center h-16">
		{#each tabs as tab}
			<li>
				<a
					href={tab.href}
					class="flex flex-col items-center gap-1 px-4 py-2 transition-colors"
					class:text-primary={isActive(tab.href)}
					class:text-muted={!isActive(tab.href)}
					aria-current={isActive(tab.href) ? 'page' : undefined}
				>
					<!-- SVG icon aquí -->
					<span class="text-xs">{tab.label}</span>
				</a>
			</li>
		{/each}
	</ul>
</nav>
```
