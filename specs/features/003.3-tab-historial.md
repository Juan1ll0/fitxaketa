## Resumen

Implementar el tab Historial (`/historial`) con listado de jornadas agrupado por dia, mostrando hora inicio, hora fin, duracion formateada e icono de sync. Migrar la funcionalidad existente de `/registros` a esta nueva ruta.

## Contexto

Actualmente existe `/registros` con un listado plano de jornadas. Esta sub-spec lo reorganiza en una vista agrupada por dia con mejor presentacion y la mueve a `/historial`.

## Historias de usuario

- Como usuario, quiero ver un historial de mis jornadas registradas, agrupadas por dia, para revisar facilmente cuanto trabaje cada dia

## Criterios de aceptacion

### Listado agrupado

- [ ] Listado de jornadas agrupado por dia
- [ ] Cabecera de grupo con fecha formateada:
  - "Hoy" si la fecha es hoy
  - "Ayer" si la fecha es ayer
  - Formato "DD MMM YYYY" para el resto (ej: "20 jun 2026")
- [ ] Grupos ordenados por fecha descendente (mas reciente primero)
- [ ] Jornadas dentro de cada grupo ordenadas por `start_time` descendente

### Cada jornada muestra

- [ ] Hora de inicio formateada (HH:MM)
- [ ] Hora de fin formateada (HH:MM), o "En curso" si `end_time` es null
- [ ] Duracion formateada en HH:MM:SS (o "En curso" si la jornada esta abierta)
- [ ] Icono de sync: ✅ si `synced === 1`, ❌ si `synced === 0`

### Estado vacio

- [ ] Mensaje si no hay jornadas: "Aun no hay fichajes registrados"
- [ ] Icono o ilustracion simple acompanando el mensaje
- [ ] Boton opcional: "Fichar ahora" que navega al tab Fichar

### Migracion

- [ ] Ruta `/historial` creada con `+page.svelte`
- [ ] Ruta `/registros` se elimina (o redirige a `/historial`)
- [ ] Referencias a `/registros` en otros archivos se actualizan (ej: enlaces en `+page.svelte` si existen)
- [ ] Tests de `/registros` se migran a `/historial` (si existen en `registros/page.test.ts`)

### Utilidades (src/lib/utils/dashboard.ts)

```typescript
export function agruparPorDia(jornadas: Jornada[]): Map<string, Jornada[]> {
	const grupos = new Map<string, Jornada[]>();
	const hoy = new Date();
	hoy.setHours(0, 0, 0, 0);
	const ayer = new Date(hoy);
	ayer.setDate(ayer.getDate() - 1);

	for (const j of jornadas) {
		const fecha = new Date(j.start_time);
		fecha.setHours(0, 0, 0, 0);

		let key: string;
		if (fecha.getTime() === hoy.getTime()) {
			key = 'Hoy';
		} else if (fecha.getTime() === ayer.getTime()) {
			key = 'Ayer';
		} else {
			key = new Intl.DateTimeFormat('es-ES', {
				day: '2-digit',
				month: 'short',
				year: 'numeric'
			}).format(fecha);
		}

		if (!grupos.has(key)) grupos.set(key, []);
		grupos.get(key)!.push(j);
	}

	return grupos;
}

export function formatearHora(date: Date): string {
	return new Intl.DateTimeFormat('es-ES', {
		hour: '2-digit',
		minute: '2-digit'
	}).format(date);
}

export function formatearDuracion(minutos: number | null): string {
	if (minutos === null) return 'En curso';
	const h = Math.floor(minutos / 60);
	const m = minutos % 60;
	return `${String(h).padStart(2, '0')}:${String(m).padStart(2, '0')}:00`;
}
```

### JornadaCard.svelte

```svelte
<script lang="ts">
	import type { Jornada } from '$lib/db';
	import { formatearHora, formatearDuracion } from '$lib/utils/dashboard';

	let { jornada }: { jornada: Jornada } = $props();

	const esAbierta = jornada.status === 'open';
</script>

<div class="flex items-center justify-between py-3 px-4 border-b border-border">
	<div class="flex items-center gap-4">
		<span class="text-text-secondary text-sm w-14">
			{formatearHora(jornada.start_time)}
		</span>
		<span class="text-text-secondary">→</span>
		<span class="text-text-secondary text-sm w-14">
			{esAbierta ? 'En curso' : formatearHora(jornada.end_time!)}
		</span>
	</div>

	<div class="flex items-center gap-3">
		<span class="text-text font-mono text-sm">
			{esAbierta ? 'En curso' : formatearDuracion(jornada.duration)}
		</span>
		<span class="text-lg" title={jornada.synced ? 'Sincronizado' : 'No sincronizado'}>
			{jornada.synced ? '✅' : '❌'}
		</span>
	</div>
</div>
```

### +page.svelte (Tab Historial)

```svelte
<script lang="ts">
	import { onMount } from 'svelte';
	import { getAllJornadas, type Jornada } from '$lib/db';
	import { agruparPorDia } from '$lib/utils/dashboard';
	import DiaGroup from '$lib/components/DiaGroup.svelte';

	export const ssr = false;

	let grupos = $state<Map<string, Jornada[]>>(new Map());
	let cargando = $state(true);

	onMount(async () => {
		const jornadas = await getAllJornadas();
		grupos = agruparPorDia(jornadas);
		cargando = false;
	});
</script>

<div class="min-h-screen bg-background">
	<div class="max-w-lg mx-auto py-4">
		{#if cargando}
			<p class="text-center text-text-secondary py-8">Cargando...</p>
		{:else if grupos.size === 0}
			<div class="text-center py-16">
				<p class="text-4xl mb-4">📋</p>
				<p class="text-text-secondary text-lg">Aun no hay fichajes registrados</p>
			</div>
		{:else}
			{#each [...grupos.entries()] as [fecha, jornadas]}
				<DiaGroup {fecha} {jornadas} />
			{/each}
		{/if}
	</div>
</div>
```
