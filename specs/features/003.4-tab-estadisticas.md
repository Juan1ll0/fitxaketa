## Resumen

Implementar el tab Estadisticas (`/estadisticas`) con: selector de periodo (Semana/Mes/Ano), grafica de barras con Chart.js mostrando horas decimales por jornada/fichaje, y resumen con total horas, media diaria, dias trabajados y numero de jornadas.

## Contexto

La app necesita mostrar estadisticas de horas trabajadas. Chart.js no esta instalado aun. Se usaran datos de Dexie (`getAllJornadas()`) filtrados por periodo.

## Historias de usuario

- Como usuario, quiero ver graficas de horas trabajadas (una barra por jornada/fichaje) para visualizar mi productividad
- Como usuario, quiero poder seleccionar el periodo (semana/mes/ano) para ver diferentes rangos de tiempo

## Criterios de aceptacion

- [ ] Botones "Semana" / "Mes" / "Ano" en la parte superior
- [ ] Default: "Mes" seleccionado
- [ ] Boton activo resaltado con color del theme
- [ ] Al cambiar periodo, la grafica y el resumen se actualizan reactivamente

### Grafica de barras

- [ ] Grafica de barras con Chart.js
- [ ] Eje X: fechas de los fichajes del periodo seleccionado
- [ ] Eje Y: horas decimales trabajadas
- [ ] Puede haber mas de una barra por dia (una por cada jornada/fichaje)
- [ ] Cada barra representa una jornada con su duracion en horas decimales
- [ ] Etiquetas del eje X legibles (formato "DD/MM" o similar)
- [ ] Grafica responsiva (se adapta al ancho de pantalla)
- [ ] Colores de la grafica usan tokens del theme (`--color-primary` para barras)
- [ ] Tooltip al pasar sobre una barra mostrando: fecha, hora inicio/fin, duracion

### Resumen

- [ ] Seccion debajo de la grafica con:
  - Total horas del periodo (suma de duraciones)
  - Media diaria (total horas / dias trabajados)
  - Dias trabajados en el periodo (dias con al menos 1 jornada)
  - Numero total de jornadas en el periodo
- [ ] Formato legible: "Total: 40h 30m | Media: 8h 6m/dia | 5 dias | 8 jornadas"
- [ ] Se actualiza reactivamente al cambiar periodo

### Datos

- [ ] Datos obtenidos de `getAllJornadas()` de Dexie
- [ ] Filtrado por periodo usando funcion utilitaria `filtrarPorPeriodo()`
- [ ] Solo se cuentan jornadas cerradas (`status === 'closed'` y `duration !== null`)

### Rendimiento

- [ ] Chart.js se carga solo en esta ruta (lazy import o code-splitting)

### Utilidades (src/lib/utils/dashboard.ts)

```typescript
export type Periodo = 'semana' | 'mes' | 'ano';

export function filtrarPorPeriodo(jornadas: Jornada[], periodo: Periodo): Jornada[] {
  const ahora = new Date();
  const inicio = new Date(ahora);

  switch (periodo) {
    case 'semana':
      inicio.setDate(inicio.getDate() - 7);
      break;
    case 'mes':
      inicio.setMonth(inicio.getMonth() - 1);
      break;
    case 'ano':
      inicio.setFullYear(inicio.getFullYear() - 1);
      break;
  }

  inicio.setHours(0, 0, 0, 0);
  return jornadas.filter(j => j.start_time >= inicio && j.status === 'closed');
}

export function calcularResumen(jornadas: Jornada[]): {
  totalHoras: number;
  mediaDiaria: number;
  diasTrabajados: number;
  totalJornadas: number;
} {
  if (jornadas.length === 0) {
    return { totalHoras: 0, mediaDiaria: 0, diasTrabajados: 0, totalJornadas: 0 };
  }

  const totalMinutos = jornadas.reduce((acc, j) => acc + (j.duration || 0), 0);
  const totalHoras = totalMinutos / 60;

  // Dias unicos
  const diasUnicos = new Set(
    jornadas.map(j => new Date(j.start_time).toDateString())
  );
  const diasTrabajados = diasUnicos.size;

  return {
    totalHoras,
    mediaDiaria: totalHoras / diasTrabajados,
    diasTrabajados,
    totalJornadas: jornadas.length
  };
}

export function formatearHorasDecimal(horas: number): string {
  const h = Math.floor(horas);
  const m = Math.round((horas - h) * 60);
  return `${h}h ${m}m`;
}
```

### StatsChart.svelte

```svelte
<script lang="ts">
  import { onMount, onDestroy } from 'svelte';
  import type { Jornada } from '$lib/db';

  let { jornadas }: { jornadas: Jornada[] } = $props();

  let canvas: HTMLCanvasElement;
  let chart: any;

  onMount(async () => {
    const { Chart, BarController, BarElement, CategoryScale, LinearScale, Tooltip, Legend } =
      await import('chart.js');

    Chart.register(BarController, BarElement, CategoryScale, LinearScale, Tooltip, Legend);

    const labels = jornadas.map(j => {
      const d = new Date(j.start_time);
      return `${String(d.getDate()).padStart(2, '0')}/${String(d.getMonth() + 1).padStart(2, '0')}`;
    });

    const data = jornadas.map(j => (j.duration || 0) / 60); // minutos a horas

    chart = new Chart(canvas, {
      type: 'bar',
      data: {
        labels,
        datasets: [{
          label: 'Horas',
          data,
          backgroundColor: 'rgba(59, 130, 246, 0.7)', // --color-primary
          borderColor: 'rgba(59, 130, 246, 1)',
          borderWidth: 1
        }]
      },
      options: {
        responsive: true,
        maintainAspectRatio: false,
        plugins: {
          legend: { display: false },
          tooltip: {
            callbacks: {
              label: (ctx) => `${ctx.parsed.y.toFixed(2)}h`
            }
          }
        },
        scales: {
          y: {
            beginAtZero: true,
            ticks: { color: '#9CA3AF' },
            grid: { color: 'rgba(255,255,255,0.1)' }
          },
          x: {
            ticks: { color: '#9CA3AF' },
            grid: { display: false }
          }
        }
      }
    });
  });

  onDestroy(() => {
    chart?.destroy();
  });

  // Actualizar grafica cuando cambian las jornadas
  $effect(() => {
    if (chart && jornadas) {
      chart.data.labels = jornadas.map(j => {
        const d = new Date(j.start_time);
        return `${String(d.getDate()).padStart(2, '0')}/${String(d.getMonth() + 1).padStart(2, '0')}`;
      });
      chart.data.datasets[0].data = jornadas.map(j => (j.duration || 0) / 60);
      chart.update();
    }
  });
</script>

<div class="w-full h-64">
  <canvas bind:this={canvas}></canvas>
</div>
```

### +page.svelte (Tab Estadisticas)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { getAllJornadas, type Jornada } from '$lib/db';
  import { filtrarPorPeriodo, calcularResumen, formatearHorasDecimal, type Periodo } from '$lib/utils/dashboard';
  import StatsChart from '$lib/components/StatsChart.svelte';

  export const ssr = false;

  let periodo = $state<Periodo>('mes');
  let todasJornadas = $state<Jornada[]>([]);
  let jornadasFiltradas = $state<Jornada[]>([]);
  let resumen = $state({ totalHoras: 0, mediaDiaria: 0, diasTrabajados: 0, totalJornadas: 0 });

  const periodos: { value: Periodo; label: string }[] = [
    { value: 'semana', label: 'Semana' },
    { value: 'mes', label: 'Mes' },
    { value: 'ano', label: 'Ano' }
  ];

  onMount(async () => {
    todasJornadas = await getAllJornadas();
    actualizarDatos();
  });

  function actualizarDatos() {
    jornadasFiltradas = filtrarPorPeriodo(todasJornadas, periodo);
    resumen = calcularResumen(jornadasFiltradas);
  }

  function cambiarPeriodo(nuevo: Periodo) {
    periodo = nuevo;
    actualizarDatos();
  }
</script>

<div class="min-h-screen bg-background px-4 py-6">
  <div class="max-w-lg mx-auto">
    <!-- Selector de periodo -->
    <div class="flex gap-2 mb-6">
      {#each periodos as p}
        <button
          onclick={() => cambiarPeriodo(p.value)}
          class="flex-1 py-2 px-4 rounded-lg font-medium transition-colors"
          class:bg-primary={periodo === p.value}
          class:text-white={periodo === p.value}
          class:bg-surface-secondary={periodo !== p.value}
          class:text-text-secondary={periodo !== p.value}
        >
          {p.label}
        </button>
      {/each}
    </div>

    <!-- Grafica -->
    <div class="bg-surface rounded-xl p-4 mb-6">
      {#if jornadasFiltradas.length > 0}
        <StatsChart jornadas={jornadasFiltradas} />
      {:else}
        <div class="h-64 flex items-center justify-center">
          <p class="text-text-secondary">No hay datos para este periodo</p>
        </div>
      {/if}
    </div>

    <!-- Resumen -->
    <div class="bg-surface rounded-xl p-4">
      <h3 class="text-text-secondary text-sm font-semibold uppercase tracking-wide mb-3">Resumen</h3>
      <div class="grid grid-cols-2 gap-4">
        <div>
          <p class="text-text-secondary text-xs">Total horas</p>
          <p class="text-text text-lg font-bold">{formatearHorasDecimal(resumen.totalHoras)}</p>
        </div>
        <div>
          <p class="text-text-secondary text-xs">Media diaria</p>
          <p class="text-text text-lg font-bold">{formatearHorasDecimal(resumen.mediaDiaria)}</p>
        </div>
        <div>
          <p class="text-text-secondary text-xs">Dias trabajados</p>
          <p class="text-text text-lg font-bold">{resumen.diasTrabajados}</p>
        </div>
        <div>
          <p class="text-text-secondary text-xs">Jornadas</p>
          <p class="text-text text-lg font-bold">{resumen.totalJornadas}</p>
        </div>
      </div>
    </div>
  </div>
</div>
```
