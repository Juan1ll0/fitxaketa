# 003.2 - Tab Fichar: Cronometro, Estado y Resumen del Dia

**Status:** approved
**Creada:** 2026-06-21
**Autor:** sistema
**Depende de:** [003.1](./003.1-navegacion.md)
**Spec padre:** [003](./003-dashboard-componentes.md)

## Resumen

Implementar el tab Fichar (home `/`) con: fecha actual en formato largo, cronometro grande centrado, indicador de estado "Trabajando"/"Descansando", boton Start/Stop reactivo, y resumen del dia (total horas + numero de jornadas).

## Contexto

El cronometro base ya funciona via el store `app-state.ts` con `getElapsed()`, `getClockedIn()`, `startJornada()`, `stopJornada()`. Esta sub-spec integra estos valores en una pantalla completa con informacion adicional.

## Historias de usuario

- Como usuario, quiero ver un cronometro que muestre cuanto llevo fichado hoy y poder iniciar/parar el fichaje
- Como usuario, quiero ver la fecha actual y un resumen de las horas trabajadas hoy

## Criterios de aceptacion

### Fecha actual

- [ ] Fecha visible en la parte superior de la pantalla
- [ ] Formato largo en espanol: "Lunes, 21 de junio de 2026"
- [ ] Se actualiza automaticamente al cambiar de dia (si la app permanece abierta)
- [ ] Funcion utilitaria `formatearFecha(date: Date): string` en `src/lib/utils/dashboard.ts`

### Cronometro

- [ ] Cronometro grande y centrado mostrando tiempo transcurrido
- [ ] Formato HH:MM:SS (reutiliza `getElapsed()` del store)
- [ ] Actualizacion reactiva cada segundo
- [ ] Tamano de fuente grande (ej: `text-6xl` o similar) para visibilidad

### Indicador de estado

- [ ] Texto visible: "Trabajando" o "Descansando"
- [ ] Color diferenciado:
  - "Trabajando": color primario/verde (`--color-primary` o token success si existe)
  - "Descansando": color secundario (`--color-text-secondary`)
- [ ] Posicion: debajo del cronometro o junto a el
- [ ] Icono opcional: punto verde parpadeante si trabajando, gris si descansando

### Boton Start/Stop

- [ ] Boton grande y accesible debajo del cronometro
- [ ] Texto reactivo:
  - "Fichar entrada" si `getClockedIn() === false` (descansando)
  - "Fichar salida" si `getClockedIn() === true` (trabajando)
- [ ] Accion:
  - Click en "Fichar entrada" llama a `startJornada()` del store
  - Click en "Fichar salida" llama a `stopJornada()` del store
- [ ] Estilo del boton:
  - "Fichar entrada": color primario (`--color-primary`)
  - "Fichar salida": color danger (`--color-danger`)
- [ ] Feedback visual al hacer click (estado hover, active)

### Resumen del dia

- [ ] Seccion debajo del boton con informacion del dia actual
- [ ] Total horas fichadas hoy (suma de `duration` de jornadas cerradas hoy + tiempo transcurrido si hay jornada abierta)
- [ ] Numero de jornadas hoy (count de jornadas con `start_time` en el dia actual)
- [ ] Formato: "Hoy: 7h 30m | 2 jornadas"
- [ ] Se actualiza reactivamente al iniciar/parar jornada
- [ ] Funcion utilitaria `calcularResumenDia(jornadas: Jornada[]): { totalHoras: number, totalJornadas: number }` en `src/lib/utils/dashboard.ts`

### Persistencia

- [ ] El cronometro persiste ante recargas (basado en `start_time` de la jornada abierta en IndexedDB)
- [ ] Al recargar con jornada activa, el cronometro continua desde el timestamp
- [ ] `export const ssr = false` (acceso a Dexie via store)

### +page.svelte (Tab Fichar)

```svelte
<script lang="ts">
	import { onMount } from 'svelte';
	import {
		initAppState,
		startJornada,
		stopJornada,
		getClockedIn,
		getElapsed,
		subscribe
	} from '$lib/stores/app-state';
	import { getAllJornadas, type Jornada } from '$lib/db';
	import { formatearFecha, calcularResumenDia } from '$lib/utils/dashboard';

	export const ssr = false;

	let clockedIn = $state(false);
	let elapsed = $state('00:00:00');
	let hoy = $state('');
	let resumenDia = $state({ totalHoras: 0, totalJornadas: 0 });
	let jornadas = $state<Jornada[]>([]);

	onMount(async () => {
		await initAppState();
		clockedIn = getClockedIn();
		elapsed = getElapsed();
		hoy = formatearFecha(new Date());

		// Suscribirse a cambios del store
		subscribe(() => {
			clockedIn = getClockedIn();
			elapsed = getElapsed();
		});

		// Cargar jornadas del dia
		await cargarJornadasHoy();
	});

	async function cargarJornadasHoy() {
		const todas = await getAllJornadas();
		const inicioHoy = new Date();
		inicioHoy.setHours(0, 0, 0, 0);
		jornadas = todas.filter((j) => j.start_time >= inicioHoy);
		resumenDia = calcularResumenDia(jornadas);
	}

	async function handleFichar() {
		if (clockedIn) {
			await stopJornada();
		} else {
			await startJornada();
		}
		await cargarJornadasHoy();
	}
</script>

<div class="flex flex-col items-center justify-center min-h-screen px-4 py-8">
	<!-- Fecha -->
	<p class="text-text-secondary text-lg mb-8">{hoy}</p>

	<!-- Cronometro -->
	<div class="text-center mb-4">
		<p class="text-6xl font-mono font-bold text-text">{elapsed}</p>
	</div>

	<!-- Estado -->
	<div class="mb-8">
		{#if clockedIn}
			<span class="text-primary font-semibold">Trabajando</span>
		{:else}
			<span class="text-text-secondary">Descansando</span>
		{/if}
	</div>

	<!-- Boton -->
	<button
		onclick={handleFichar}
		class="px-8 py-4 rounded-lg font-semibold text-lg transition-colors"
		class:bg-primary={clockedIn}
		class:hover:bg-primary-dark={clockedIn}
		class:bg-danger={!clockedIn}
		class:hover:bg-danger-dark={!clockedIn}
		class:text-white
	>
		{clockedIn ? 'Fichar salida' : 'Fichar entrada'}
	</button>

	<!-- Resumen del dia -->
	<div class="mt-8 text-center text-text-secondary">
		<p class="text-sm">
			Hoy: {Math.floor(resumenDia.totalHoras)}h {Math.round((resumenDia.totalHoras % 1) * 60)}m | {resumenDia.totalJornadas}
			{resumenDia.totalJornadas === 1 ? 'jornada' : 'jornadas'}
		</p>
	</div>
</div>
```

### Utilidades (src/lib/utils/dashboard.ts)

```typescript
import type { Jornada } from '$lib/db';

export function formatearFecha(date: Date): string {
	return new Intl.DateTimeFormat('es-ES', {
		weekday: 'long',
		year: 'numeric',
		month: 'long',
		day: 'numeric'
	}).format(date);
}

export function calcularResumenDia(jornadas: Jornada[]): {
	totalHoras: number;
	totalJornadas: number;
} {
	const totalMinutos = jornadas.reduce((acc, j) => acc + (j.duration || 0), 0);
	return {
		totalHoras: totalMinutos / 60,
		totalJornadas: jornadas.length
	};
}
```
