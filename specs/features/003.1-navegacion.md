## Resumen

Implementar el componente BottomNav con 3 tabs (Fichar, Historial, Estadisticas) y actualizar el layout raiz para incluir navegacion fija inferior con padding adecuado.

## Contexto

Actualmente la app tiene dos rutas sueltas (`/` y `/registros`) sin navegacion unificada. Se necesita un bottom nav tipo app movil que permita moverse entre secciones.

## Historias de usuario

- Como usuario, quiero una navegacion por tabs inferior para moververme entre las secciones de la app sin recargar la pagina

### BottomNav.svelte

```svelte
<script lang="ts">
	import { page } from '$app/stores';

	const tabs = [
		{ href: '/', label: 'Fichar', icon: 'clock' },
		{ href: '/historial', label: 'Historial', icon: 'list' },
		{ href: '/estadisticas', label: 'Estadisticas', icon: 'chart' }
	];

	function isActive(href: string): boolean {
		if (href === '/') return $page.url.pathname === '/';
		return $page.url.pathname.startsWith(href);
	}
</script>
```
